PMA transfer function evaluation

This viz examines the transfer function of particle mass analyzers like the centrifugal particle mass analyer (CPMA) or aerosol particle mass analyzer (APM).

We refer the reader to Sipkens et al. (2020) for more information on the underlying analytical functions. A supporting repository, including Python code for evaluating the PMA transfer function is available here. A Matlab version is also available here.

The default view shows contributions from uncharged particles as well as the sum over integer charge states z = 1 through 3. Note that Case 1S has an anomaly for z = 1 that is a consequence of solve for the equilibrium raidus, which appears as a linear increase in the transfer function for m/m* > 2 and demonstrates a limitation of that approach.
